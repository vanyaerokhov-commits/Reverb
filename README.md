# Reverb — Music Events App

A web application for discovering concerts, buying tickets, exploring artists, and shopping official merch.

**Live site:** https://reverb-ten.vercel.app

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router v7
- shadcn/ui

## Running locally

```bash
npm i
npm run dev
```

Opens at `http://localhost:5173`
